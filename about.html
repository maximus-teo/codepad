About Codepad

Codepad is an online code editor and compiler, which:

  • supports widely-used programming languages like Java, Python and C++,
  • provides custom formatting options for code editor and output console,
  • has dark mode and light mode support,
  • automatically saves your code as you type, and
  • has copy and download functionalities.

APIs: Monaco Editor (with Judge0 CE)
Languages & Frameworks: JavaScript (front-end), Node.js (back-end), HTML and CSS.
